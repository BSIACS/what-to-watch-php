## Запуск
Развёртывание приложения выполняется следующими командами:

Cоздать необходимые образы docker-контейнеров

```bash
sudo make build
```

Запустить docker-контейнеры

```bash
sudo make start
```

Выполнить миграцию базы данных

```bash
sudo make migrate
```

Заполнить базу данных тестовыми данными

```bash
sudo make seed
```
